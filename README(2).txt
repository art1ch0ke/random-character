Создайте character.css и characker.js файлы.
Подключите к index.html и убедитесь, что они
правильно работают
Найдите на рабочем столе папку images и
вставьте в проект.


HTML
Под кнопкой создайте div
с классом container.
Внутри него div с id character-box.
Внутри character-box создайте параграф 
с id character-text
и img с id character-image с пустым src.
Под character-box внутри container 
вставьте кнопку с id generate и напишите
"Создать персонажа" или "Create a character".

CSS
Напишите вместе CSS код вместе с преподователем, 
добавив стили для body, container, generate,
character-box, character-text и character-image

По очереди напишите стили для анимацией
#character-box.show
#character-box.hide
#character-box.glow

SCRIPT 
Создайте массив names с именами для персонажей
Используйте свое имя и добавьте к нему "Edward",
"Morgan", "Leon", "Sophee", "Adelina"
Создайте массив classes с классами 
"warrior", "mage", "archer", "thief"
Создайте массив weopens с оружием 
Sword, Bow, Dagger и Staff

Создайте объект images,
в котором ключами будут названия классов, 
а значениями адреса к картинкам.

Навесьте на кнопку generate обработчик события click.

Создайте переменную output и свяжите
c id character-text
Создайте переменную image и свяжите с 
элементом character-image по id 
characterBox для id character-box.
